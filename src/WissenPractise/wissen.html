It will contain the values [3, 5, 7]. arguments object is not a real array, so it doesn't have array methods like forEach or map, but it can be iterated through using a loop. The arguments object allows you to access all the passed values, even if the function is defined with a specific number of parameters. better recommended alternate: function addNumbers(...numbers) { console.log(numbers); } addNumbers(3, 5, 7); //numbers is an array containing the values [3, 5, 7] Arrow functions in JavaScript do not have their own arguments object. Unlike regular functions, arrow functions do not bind their own this, arguments, super, or new.target. Instead, they inherit these values from the enclosing lexical context.
